
# Comunity Blog

Este foi um projeto desenvolvido para aplicar conhecimentos da disciplina de arquitetura de software, implementando um BaaS (Backend as a Service), com supabase.

<br>

## Funcionalidades

- **Autenticação**: Permite criar um conta com ativação por email, e logar no sistema.
- **Realizar postagem**: Crie postagens para que outros usuários possam ler suas ideias.
- **Curtir e descurtir postagens**: É possível aumentar ou diminuir o engajamentos com as curtidas.

<br>

## Modo de uso

- Ao criar sua conta através do botão cadastrar, é necessário realizar a ativação da conta por email.
- Na tela inicial, clique em criar post e insira o título e a descrição.
- é possível interagir com os posts através de curtidas nos botões de like e dislike.
- Realize o logout do sistema através do botão de logout no canto superior direito.
